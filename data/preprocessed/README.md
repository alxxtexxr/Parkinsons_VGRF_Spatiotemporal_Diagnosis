# Preprocessed Data
## Naming Convention
- `Ga`, `Ju`, or `Si`: The source study of the dataset.
- `k`: The number of folds used in K-Folds Cross-Validation.
- `w`: The window size.
- `s`: The stride size.
- `f`: The number of features.
- `w_anomaly`: With anomaly data.
- `wo_synth`: Without synthetic data.
